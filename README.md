# URL Shortener

A simple URL shortener built with Node.js and Express. This project allows you to create shortened URLs for any long URL you provide.

## Features

- Create shortened URLs
- Custom URL codes (optional)
- Persistent storage using JSON file
- Simple and clean interface
- Responsive design

## Setup

1. Clone the repository:

```bash
git clone https://github.com/yourusername/url-shortener.git
cd url-shortener
```

2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
npm start
```

4. Open your browser and visit `http://localhost:3002`

## Usage

1. Enter the long URL you want to shorten
2. (Optional) Enter a custom code for the shortened URL
3. Click "Shorten URL"
4. Copy the generated shortened URL
